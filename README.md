## ⛩️ Origin Story

<img align="right" width="280" src="https://media.giphy.com/media/qgQUggAC3Pfv687qPC/giphy.gif" />

> *"Every great developer you know got there by solving problems they were unqualified to solve — until they actually did it."*

My name is **Lameck** — but on the internet, I'm **Stellen**.

I didn't take the traditional route. No bootcamp. No university. Just a laptop, an internet connection, and a hunger to build things that matter.

I started with pure curiosity — tinkering with HTML, breaking things, fixing them, and slowly falling deeper into the rabbit hole. I taught myself everything: **from HTML fundamentals all the way to backend architecture, Docker, and deployment pipelines.**

A friend helped shape my path early on — but the drive? That was always mine.

I'm not waiting for a degree to validate what I can do. Every project I ship is proof.

**South Africa 🇿🇦 made me. The internet raised me. Code is how I speak.**
